Apache Isis™ is a platform to let you rapidly develop domain-driven apps in Java.

The Creation Tool Prototype demonstrates how a maritime ontology and taxonomy is used to make consistent machine-readable regulations to support compliance and enforcement.

This prototype is based on the Isis Addons' Summernote Editor (Wicket) component, configured for a very simple and purposefully minimal application.

For more details on Summernote, see the github repo.

For more details on the CreationTool GUI, see the github repo.
